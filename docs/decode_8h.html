int l8w8jwt_decode ( struct l8w8jwt_decoding_params *  params,
enum l8w8jwt_validation_result *  out 
)

Decode (and validate) a JWT using specific parameters.

The resulting l8w8jwt_validation_result written into the passed "out" pointer contains validation failure flags (see the l8w8jwt_validation_result enum docs for more details).

If validation succeeds, the l8w8jwt_validation_result receives the value 0 (enum value L8W8JWT_VALID).

Parameters
paramsThe parameters to use for decoding and validating the token.
outWhere to write the validation result flags into (0 means success).
